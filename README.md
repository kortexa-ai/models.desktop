## Quick Start

```bash
npm install
npm run setup
npm run electron:dev
```

## Scripts

- `npm run setup` - Initial setup (build + add Electron platform)
- `npm run electron:dev` - Build and run the Electron app
- `npm start` - Run Vite dev server (web only)
