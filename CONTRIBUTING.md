# Contributing to models.desktop

We welcome contributions! However, before we can accept any contributions, you must sign our Contributor License Agreement (CLA).

## Contributor License Agreement

**CLA signing process is currently being finalized.** Once available, instructions will be posted here.

Until the CLA process is in place, we cannot merge external contributions. Feel free to open issues for bugs or feature requests in the meantime.

## How to Contribute (once CLA is signed)

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Ensure tests pass and code follows project conventions
5. Commit your changes with clear, descriptive messages
6. Push to your fork
7. Open a Pull Request

## Questions or Suggestions

If you have questions or suggestions, please feel free to open an issue.
